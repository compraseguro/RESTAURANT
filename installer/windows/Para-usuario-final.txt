Instalación completada

El servicio de impresión Resto-FADEY ya está instalado y se iniciará solo cada vez que encienda este equipo.

Qué hacer ahora:
1. Abra la aplicación Resto-FADEY en el navegador (Chrome o Edge recomendado).
2. Vaya al menú lateral → Impresora (caja, cocina o bar según corresponda).
3. Elija cómo está conectada cada impresora (red, USB desde el navegador, o Windows) y guarde.

Si una estación no imprime o http://127.0.0.1:3049/health dice «conexión rechazada»:
— El programa local no está en marcha. Pruebe en este orden:
  1) Menú Inicio → carpeta Resto-FADEY → «Resto-FADEY impresión (inicio manual)» (deja el servicio en segundo plano).
  2) Vuelva a abrir http://127.0.0.1:3049/health en el navegador.
  3) Si sigue fallando: abra «Programador de tareas» de Windows, busque la tarea «RestoFadeyPrintService»,
     ejecútela manualmente y revise «Historial» por errores.
  4) Revise el archivo de aviso del instalador (si se generó):
     %LOCALAPPDATA%\RestoFadey\print-service-install.txt
— Instale iniciando sesión con el MISMO usuario de Windows que usará Chrome o Edge (con varios usuarios, cada uno puede necesitar su instalación).
— Vuelva a ejecutar el instalador .exe si faltan archivos en la carpeta del programa (p. ej. Run-PrintService.ps1).
